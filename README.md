# sourdough-coin
Sourdough Coin (SOURD) is a purpose-built ERC-20 cryptocurrency that blends blockchain innovation with real-world community impact. The project funds weekly fresh bread donations for local food pantries while creating sustainable employment opportunities for individuals seeking a new start after incarceration.

Our mission is simple:
Use crypto to feed communities, restore dignity, and build opportunities.

With a limited 1,000,000 token supply, transparent on-chain operations, and a mission-backed utility model, Sourdough Coin empowers every holder to directly contribute to meaningful, measurable change.
